JavaScript Counter App

Features:
*   Increase counter
*   Decrease counter
*   Reset counter to zero
*   Simple and beginner-friendly code

Technologies Used:<br>
*HTML<br>
*JavaScript<br><br>

How to Run the Project:<br>
1.Download or clone the repository<br>
git clone https://github.com/mosley217/counter-app.git<br>
2.Open the project folder<br>
3.Double-click index.html to open in your browser<br><br>

Operational Overview:<br>
The JavaScript application retrieves the counter value through the use of document.getElementById().<br>
Event listeners are implemented to detect button clicks.<br>
The counter value is updated dynamically.

Preview:
The application features a user-friendly interface with five functional buttons: Increase, Decrease, Reset, Save, and Load.
